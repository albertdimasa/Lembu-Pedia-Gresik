## Cara Install

1. Clone Project 
```bash
  git clone https://github.com/albertdimasa/sapi-app.git
```

2. Install Composer (pastikan sudah install composer)
```bash
  composer install
  composer update
```

3. Buat Database dan sesuaikan namanya dengan di file .env
```bash
  php artisan migrate
  php artisan db:seed
```
